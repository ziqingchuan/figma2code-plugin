## AI优化流程
AI优化流程时序图mermaid代码：

```mermaid
sequenceDiagram
    participant 代码生成模块 as 代码生成模块
    participant 解析器 as HTML解析器
    participant Coze as Coze API
    participant 重组器 as HTML重组器

    Note over 代码生成模块: 开始AI优化
    代码生成模块->>解析器: parseHTMLToNodes()
    Note over 解析器: 提取节点结构<br/>样式映射<br/>SVG内容
    解析器-->>代码生成模块: 返回节点数据

    代码生成模块->>Coze: 发送HTTP请求
    Note over Coze: 调用Coze工作流<br/>生成语义化类名
    alt AI调用成功
        Coze-->>代码生成模块: 返回类名映射
        Note over 代码生成模块: 检查数量一致性
        alt 数量一致
            Note over 代码生成模块: 使用AI类名
        else 数量不一致
            Note over 代码生成模块: 使用人工类名
        end
    else AI调用失败
        Note over 代码生成模块: 使用回退方案
    end

    代码生成模块->>重组器: parseNodesToHTML()
    Note over 重组器: 应用类名映射<br/>重组HTML
    重组器-->>代码生成模块: 返回优化后的HTML

    Note over 代码生成模块: 返回HTML代码
```
